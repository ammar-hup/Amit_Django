## 3. Using Context Data Effectively (30 min)

> **Goal**: Send **smart data** from views → templates

### Update `views.py`
```python
from django.shortcuts import render, get_object_or_404
from django.utils import timezone
from .models import Post

def post_list(request):
    posts = Post.objects.filter(
        published=True
    ).order_by('-created_at')
    
    context = {
        'posts': posts,
        'today': timezone.now().date(),
        'total_posts': posts.count(),
    }
    return render(request, 'blog/post_list.html', context)

def post_detail(request, pk):
    post = get_object_or_404(Post, pk=pk)
    
    context = {
        'post': post,
        'reading_time': len(post.content.split()) // 200 + 1,  # words per minute
    }
    return render(request, 'blog/post_detail.html', context)
```

### Use in `post_list.html`
```html
<p class="text-muted">Total posts: <strong>{{ total_posts }}</strong> | Today: {{ today|date:"d M Y" }}</p>
```

### Use in `post_detail.html`
```html
<p class="text-muted">
    Approx. {{ reading_time }} min read
</p>
```

## 4. Best Practices in UI Design (30 min)

> **Goal**: Teach **clean, readable, scalable** design

### 1. **Semantic HTML**
```html
<article>, <header>, <main>, <footer>
```

### 2. **Consistent Spacing**
```html
mt-5, mb-4, p-4 → Bootstrap spacing classes
```

### 3. **Visual Hierarchy**
- `h1` → main title
- `h3` → post title
- `small text-muted` → metadata

### 4. **Accessibility**
```html
alt="", lang="en", proper contrast
```

### 5. **Mobile-First**
```html
col-md-8 → full width on mobile, 8/12 on desktop
```
